# .dotfiles

Dotfiles managed using [chezmoi](https://github.com/twpayne/chezmoi).

## Quickstart

chezmoi [Quick start](https://www.chezmoi.io/quick-start/)

```bash
# install chezmoi
sh -c "$(curl -fsLS get.chezmoi.io)"

# init
chezmoi init
# add file
chezmoi add <file>

# restore
chezmoi -v apply
```
